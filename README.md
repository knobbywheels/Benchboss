# Benchboss
A mobile-first web application designed for ice hockey coaches to track real-time player statistics and "Time on Ice" (TOI) during games. The app prioritizes speed, offline reliability, and low-friction authentication to ensure coaches stay focused on the game, not the screen.

## Run Locally
**Prerequisites:**  Node.js

1. Install dependencies:
   `npm install`
2. Set the `GEMINI_API_KEY` in [.env.local](.env.local) to your Gemini API key
3. Run the app:
   `npm run dev`
